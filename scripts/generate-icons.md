# Генерация иконок для PWA

Для работы PWA нужны иконки в формате PNG:
- `public/icon-192x192.png` (192x192 пикселей)
- `public/icon-512x512.png` (512x512 пикселей)

Вы можете:
1. Использовать онлайн-генератор (например, https://realfavicongenerator.net/)
2. Создать иконки в графическом редакторе
3. Использовать SVG иконку `public/icon.svg` как основу

Иконки должны быть квадратными и содержать логотип "L" или другое изображение, связанное с LexiFlow.
